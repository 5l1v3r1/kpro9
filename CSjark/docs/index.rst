Welcome to CSjark's documentation!
==================================

CSjark is a tool for generating Lua dissectors from C struct definitions to use with Wireshark. `Wireshark <http://www.wireshark.org>`_ is a leading tool for capturing and analysing network traffic. CSjark provides a way to display the contents of the C struct data coming from an IPC packet in human-readable form in Wireshark.

You can find more about CSjark functionality in the :ref:`intro` section.
