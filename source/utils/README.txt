This folder includes utilities which might be useful for our program.

* cpp.exe is a C preprocessor for Windows. It was compiled from the LCC
  distribution <http://www.cs.princeton.edu/software/lcc/>. It is provided
  under LCC's license terms, which can be found in lcc.CPYRIGHT.txt.

* fake_libc_include comes from pycparser, they are standard C header files
  that contain only the bare necessities to allow valid parsing of the files
  that use them. It is provided under pycparser's license terms, which can be
  found in pycparser.LICENSE.
